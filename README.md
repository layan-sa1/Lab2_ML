# Lab2_ML

## Dataset
Hotel Booking Cancellation Prediction dataset.
## Problem Description
The objective of this project is to predict the booking status of hotel reservations based on customer and booking-related information.
The model will learn patterns from features such as lead time, room type, and average price to estimate whether a booking is likely to be canceled or not.
This is a supervised classification problem using a structured tabular dataset.
The target variable is booking status.
